# Idées de design — NEXYTAL Gestion

## Approche choisie : Corporate Precision Dark

**Design Movement :** Corporate Tech / SaaS Dashboard — inspiré des interfaces d'administration B2B haut de gamme (Linear, Vercel, Notion).

**Core Principles :**
- Clarté absolue : chaque élément a une raison d'être, aucun bruit visuel.
- Hiérarchie forte : titres imposants, données secondaires discrètes.
- Cohérence inter-sites : chaque site NEXYTAL a une couleur d'accent distincte mais partage la même grille.
- Efficacité opérationnelle : accès aux données en 2 clics maximum.

**Color Philosophy :**
- Fond principal : `#0F1117` (noir profond, professionnel)
- Sidebar : `#161B22` (légèrement plus clair)
- Cards : `#1C2128` (surface élevée)
- Accent global : `#2563EB` (bleu NEXYTAL)
- Couleurs par site :
  - Alt Formation : `#7C3AED` (violet — formation/savoir)
  - Nexytal Medical : `#059669` (vert — santé)
  - Nexytal Recrutement : `#2563EB` (bleu — tech/IT)
  - Nexytal Carrière : `#D97706` (ambre — carrière/ambition)
  - Nexytal Coaching : `#DC2626` (rouge — énergie/coaching)
  - Nexytal Trainer : `#0891B2` (cyan — formation pro)

**Layout Paradigm :**
- Sidebar fixe à gauche (64px collapsed / 240px expanded)
- Zone principale avec header sticky par page
- Grille asymétrique : liste + panneau détail côte à côte

**Signature Elements :**
- Badge coloré par site dans la sidebar et les en-têtes
- Indicateurs de statut animés (points pulsants)
- Séparateurs fins avec gradient

**Typography System :**
- Display : `Space Grotesk` (700) — titres et KPIs
- Body : `Inter` (400/500) — texte courant
- Mono : `JetBrains Mono` — IDs, codes, dates techniques
